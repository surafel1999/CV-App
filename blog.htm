Using Scalable Units for Responsive Site.

Yes, it does. Because %, em, rem all are scalable units which are translated by the user's browser into pixel values. Since, these are scalable units values are not always exactly the same pixel, which depends on the current browser setting.

Therefore, if the user wants to increase the size of the text, this can be done simply by using relative size on the developer's part and the browser handles the rest.

Furthermore, when websites are developed it should target many devices specially, mobile phones. In a nutshell, in this era of technology no matter what the device, the website should look great. This is done by using relative size.
